## 一、总则

### 1.1 编制目的

为有效预防、及时控制和妥善处理托管期间的各类突发事件,保障学生生命安全,最大限度地减少突发事件造成的损失,特制定本预案。

### 1.2 工作原则

- **以人为本,生命至上**:把保障学生生命安全放在首位
- **预防为主,防范在先**:立足防范,抓早抓小
- **快速反应,科学应对**:第一时间响应,科学处置
- **统一指挥,分级负责**:明确责任,各司其职

### 1.3 适用范围

本预案适用于本托管机构托管期间发生的各类突发事件。

## 三、常见突发事件应急预案

### 3.1 学生意外伤害

#### 3.1.1 轻微擦伤、碰伤

**处置流程**：
1. 立即查看伤情
2. 用碘伏消毒,贴创可贴
3. 安抚学生情绪
4. 通知家长(电话或微信)
5. 填写《事故记录表》
6. 观察后续情况

**责任人**：当值老师

**预防措施**：
- 桌角安装防撞条
- 及时清理地面杂物
- 教育学生不追逐打闹

#### 3.1.2 出血较多、扭伤、疑似骨折

**处置流程**：
1. **立即采取急救措施**：
   - 出血:用干净纱布压迫止血,抬高伤肢
   - 扭伤:冷敷,避免活动
   - 疑似骨折:固定伤处,不要移动
2. **拨打120**,说明情况和地址
3. **立即通知家长**,告知受伤情况和处置措施
4. **保留现场证据**:拍照,调取监控
5. 等待救护车,准备陪同就医
6. 填写《事故记录表》,详细记录
7. **启动保险理赔**程序

**责任人**：当值老师+负责人

**注意事项**：
- 不要随意移动受伤学生
- 保持冷静,安抚学生和其他孩子情绪
- 如实告知家长,不隐瞒

### 3.2 异物卡喉(窒息)

**识别信号**：
- 突然不能说话或哭泣
- 用手抓住喉咙
- 脸色发紫
- 呼吸困难

**处置流程**：

**1岁以上儿童 - 海姆立克急救法**：
1. 立即实施海姆立克急救法:
   - 站在孩子身后
   - 一手握拳,拳眼对准肚脐上方
   - 另一手抱住拳头
   - 快速向内向上冲击5-8次
   - 直到异物排出
2. 同时让其他老师拨打120
3. 如孩子昏迷,立即实施心肺复苏
4. 救护车到达前持续急救
5. 通知家长
6. 陪同就医
7. 填写《事故记录表》

**预防措施**：
- 禁止提供坚果、果冻等易卡喉食物
- 进食时禁止嬉戏打闹
- 教育学生细嚼慢咽
- 定期培训急救技能

**责任人**：当值老师(必须掌握海姆立克急救法)

### 3.3 过敏反应

**识别信号**：
- 皮肤红肿、瘙痒、出疹
- 呼吸困难、喘息
- 恶心、呕吐
- 严重时:休克、昏迷

**处置流程**：
1. **立即停止接触过敏源**(如正在进食,立即停止)
2. **观察症状严重程度**：
   - 轻微(仅皮肤红肿):用清水冲洗,通知家长
   - 严重(呼吸困难、呕吐):立即拨打120
3. **通知家长**,告知过敏情况
4. 如家长提供过敏药物,在家长指导下用药
5. 陪同就医(严重情况)
6. 填写《事故记录表》

**预防措施**：
- 建立学生过敏档案(查阅《学生信息登记表》)
- 提供食物前核对过敏史
- 禁止提供学生过敏的食物
- 与家长确认安全食物清单

**责任人**：当值老师

### 3.4 发热、呕吐、腹泻

**处置流程**：
1. **立即隔离**学生,安置在单独区域
2. **测量体温**:
   - 37.3°C以下:观察,多喝水
   - 37.3-38°C:通知家长接回
   - 38°C以上:立即通知家长,建议就医
3. **呕吐/腹泻**:
   - 清理呕吐物,消毒
   - 观察学生精神状态
   - 立即通知家长接回
4. **其他学生**:观察健康状况,预防传染
5. **环境消毒**:对接触区域进行消毒
6. 填写《事故记录表》

**预防措施**：
- 每日晨检,观察学生健康状况
- 保持环境卫生,定期消毒
- 发现传染病及时隔离

**责任人**：当值老师

### 3.5 学生走失

**处置流程**：
1. **立即点名确认**缺失学生
2. **快速查找**:
   - 在托管点内全面搜索
   - 询问其他学生
   - 查看监控录像
3. **立即通知家长**和紧急联系人
4. **报警**(110),提供学生照片和详细信息
5. **保留证据**:调取监控,记录时间线
6. **通知相关部门**:社区、物业协助查找
7. 找到后送医检查,安抚情绪
8. 调查原因,追究责任

**预防措施**：
- 严格接送制度,只允许授权人员接走
- 每30分钟点名1次
- 大门随时关闭,防止学生私自外出
- 安装监控设备

**责任人**：当值老师+负责人

**事故责任**：
- 因管理不善导致走失,托管机构承担全部责任
- 因非授权人员冒领导致走失,立即报警,追究法律责任

### 3.6 学生被陌生人冒领

**处置流程**：
1. **严格核对身份**:
   - 对照授权接送人名单
   - 陌生人接送必须电话确认家长
   - 查看身份证并拍照留存
2. **家长确认后方可交接**
3. **如无法联系家长**:
   - 拒绝交接
   - 联系紧急联系人
   - 必要时报警
4. **如发现异常**(如强行带走):
   - 立即制止
   - 报警(110)
   - 保护学生安全

**预防措施**：
- 建立授权接送人名单,家长签字确认
- 陌生人接送必须电话确认
- 接送时查看接送卡或身份证

**责任人**：当值老师

### 3.7 火灾

**处置流程**：
1. **发现火情**:
   - 小火:立即使用灭火器扑灭
   - 大火:立即拨打119,不要扑救
2. **立即组织疏散**:
   - 按疏散路线快速撤离
   - 带领学生到安全区域(小区广场等)
   - 清点人数,确认全部撤离
3. **通知家长**
4. **配合消防部门**调查

**疏散路线**：
- 主要路线:___________(如:客厅→大门→楼梯→小区广场)
- 备用路线:___________(如:客厅→阳台→邻居家→楼梯)

**集合地点**：___________(如:小区南门广场)

**疏散要求**：
- 用湿毛巾捂住口鼻
- 低姿前行
- 不乘坐电梯
- 不返回火场

**预防措施**：
- 定期检查电路,不超负荷用电
- 禁止使用明火
- 配备灭火器,定期检查
- 每学期组织1次消防演练

**责任人**：当值老师+全体工作人员

### 3.8 地震

**处置流程**：
1. **地震发生时**:
   - 保持冷静,不要慌乱
   - 就近躲避:桌子下、墙角、卫生间等
   - 保护头部
   - 远离窗户、吊灯等
2. **震后疏散**:
   - 快速撤离到空旷区域(小区广场)
   - 清点人数
   - 通知家长
3. **如有伤员**:
   - 立即救治
   - 拨打120

**预防措施**：
- 固定高大家具,防止倾倒
- 准备应急包(水、食物、手电筒)
- 定期组织演练

**责任人**：当值老师+全体工作人员

### 3.9 学生打架冲突

**处置流程**：
1. **立即制止**,分开双方
2. **检查是否受伤**:
   - 无伤:安抚情绪
   - 受伤:按意外伤害处理
3. **了解原因**,教育引导
4. **通知双方家长**,说明情况
5. 填写《事故记录表》

**预防措施**：
- 及时发现苗头,提前介入
- 教育学生友好相处,用语言解决问题
- 不让学生单独相处(矛盾双方)

**责任人**：当值老师

### 3.10 停电、停水

**处置流程**：
1. **保持冷静**,安抚学生
2. **使用应急照明**(手电筒、应急灯)
3. **联系物业**了解情况
4. **通知家长**,视情况提前接回
5. 如长时间停电/停水,暂停服务

**预防措施**：
- 准备应急照明设备
- 储备饮用水

**责任人**：当值老师

## 五、事故报告与调查

### 5.1 报告制度

**一般事故**(轻微擦伤):
- 及时通知家长
- 填写《事故记录表》

**较大事故**(出血、扭伤、疑似骨折):
- 立即通知家长
- 填写《事故记录表》
- 24小时内向教育部门报告(如要求)

**重大事故**(昏迷、窒息、严重外伤、死亡):
- 立即拨打120、110
- 立即通知家长
- 立即向教育部门、公安部门报告
- 保留所有证据
- 配合调查

**报告内容**：
- 事故时间、地点
- 受伤学生信息
- 事故经过
- 已采取措施
- 当前状况

### 5.2 事故调查

**调查原则**：
- 实事求是,不隐瞒不夸大
- 查明原因,分清责任
- 总结教训,防止再发生

**调查内容**：
- 事故发生原因
- 是否存在管理漏洞
- 相关人员是否尽责
- 改进措施

### 5.3 善后处理

**医疗救治**：
- 积极配合治疗
- 承担相应医疗费用(按协议和法律规定)

**心理疏导**：
- 安抚受伤学生及其他学生情绪
- 必要时请专业心理咨询师

**责任认定**：
- 因托管机构管理不善:托管机构承担责任
- 因学生自身原因:按协议约定
- 因第三方原因:向第三方追责

**保险理赔**：
- 及时启动理赔程序
- 协助家长完成理赔

**家长沟通**：
- 如实告知情况
- 诚恳道歉(如有责任)
- 商讨赔偿方案

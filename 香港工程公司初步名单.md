## 如何获取完整数据

### 方案1: 使用提供的爬虫脚本（推荐）

本仓库已包含完整的Google Maps爬虫脚本 `scrape_hk_engineering.py`，您可以在本地环境运行：

#### 运行步骤：

1. **安装依赖**
```bash
pip install playwright
playwright install chromium
playwright install-deps chromium  # 安装系统依赖
```

2. **运行脚本**
```bash
python scrape_hk_engineering.py
```

3. **获取结果**
脚本将自动：
- 搜索5个关键词（工程公司、建筑工程、机电工程、土木工程、装修工程）
- 每个关键词提取至少100家公司
- 自动去重
- 生成 `香港工程公司列表.md` 和 `香港工程公司列表.json`

#### 预计结果：
- **提取数量**: 200-500家工程公司
- **运行时间**: 30-60分钟（取决于网络速度）
- **数据内容**: 公司名称、地址、联系电话

## 常见问题

### Q: 为什么爬虫在某些环境无法运行？
A: Google Maps在某些地区可能有访问限制。建议：
- 使用VPN
- 在香港本地网络环境运行
- 使用代理服务器

### Q: 如何提高数据提取质量？
A:
- 增加滚动次数（修改 `max_scrolls` 参数）
- 增加延迟时间（修改 `await asyncio.sleep()` 时间）
- 使用更精确的关键词

### Q: 数据会重复吗？
A: 不会。脚本内置去重机制，相同公司名称只会保留一次。

### Q: 可以修改搜索关键词吗？
A: 可以。编辑脚本中的 `keywords` 列表：
```python
keywords = [
    "工程公司 香港",
    "建筑工程 香港",
    # 添加更多关键词...
]
```

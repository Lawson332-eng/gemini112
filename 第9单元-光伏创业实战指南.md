## 9.1 光伏创业方向选择与商业模式设计

### 9.1.1 光伏创业赛道全景图

#### 赛道一：分布式光伏EPC总包商

**业务定位**
- 为工商业、户用客户提供光伏电站设计、采购、施工一体化服务
- 项目规模：10kW-5MW
- 客户类型：工厂、商业综合体、学校、医院、农村居民

**商业模式**
```
收入来源：
├─ 工程收入：系统单价 × 装机容量
├─ 运维服务费：30-50元/kW/年
└─ 供应链差价：组件、逆变器采购差价

成本结构：
├─ 设备采购成本：60-65%
├─ 施工人工成本：15-20%
├─ 销售费用：8-12%
├─ 管理费用：5-8%
└─ 财务费用：2-5%（如有垫资）

典型利润率：
- 工商业项目：净利润率 8-12%
- 户用项目：净利润率 10-15%
```

**准入门槛**
- 初始投资：30-100万元
- 必备资质：电力工程施工资质（可先挂靠）
- 核心团队：3-5人（销售、设计、施工各1-2人）
- 供应商关系：至少2家组件/逆变器供应商

**适合人群**
- 有电力、建筑工程背景
- 本地人脉资源丰富
- 能接受6-12个月账期

**风险与挑战**
| 风险点 | 应对策略 |
|--------|----------|
| 垫资压力大 | 严格控制付款节点，30%-30%-30%-10% |
| 应收账款风险 | 选择信用好的客户，必要时要求保函 |
| 价格竞争激烈 | 差异化服务（快速响应、优质运维） |
| 技术迭代快 | 与头部品牌保持合作，及时更新方案 |

#### 赛道二：光伏电站投资运营商

**业务定位**
- 自建自持光伏电站，赚取长期发电收益
- 或：开发电站到一定阶段后转让，赚取开发溢价

**商业模式**

**模式A：长期持有**
```
项目案例：5MW地面光伏电站

初始投资：1500万元
├─ 设备及安装：1200万元
├─ 土地租金（20年）：150万元
├─ 并网及配套：100万元
└─ 开发费用：50万元

年度收益：
├─ 发电量：650万kWh
├─ 上网电价：0.35元/kWh
├─ 年收入：227.5万元
├─ 运维成本：15万元
└─ 净收益：212.5万元

财务指标：
- 静态回收期：7.1年
- IRR：12.3%
- 25年总收益：5312万元
```

**模式B：开发转让**
```
项目流程：
1. 获取项目批文（3-6个月）
   - 土地预审
   - 发改委备案
   - 电网接入意见函

2. 建设至30%进度（3个月）
   - 完成桩基和支架
   - 组件采购到位

3. 转让给专业电站运营商
   - 转让价格：1.2-1.5倍投资额
   - 开发周期：6-9个月
   - 投资收益率：20-50%
```

**准入门槛**
- 初始投资：500万-5000万元
- 核心能力：项目开发能力（拿地、拿批文）
- 资源要求：政府关系、土地资源
- 风险承受：较高（政策变化、电站转让不确定性）

**适合人群**
- 有资金实力的投资人
- 有政府资源或土地资源
- 能接受中长期投资回报

#### 赛道三：光伏产品经销代理商

**业务定位**
- 代理销售组件、逆变器等光伏产品
- 服务下游安装商、工程商

**商业模式**
```
合作模式：
├─ 省级代理：年任务10-30MW，区域独家
├─ 市级代理：年任务3-10MW，市级保护
└─ 县级经销：年任务1-3MW，灵活拿货

收入来源：
├─ 产品销售差价：0.05-0.15元/瓦
├─ 年度返点：完成任务后3-8%返利
├─ 项目推荐费：介绍项目给厂家，收取1-2%
└─ 技术服务费：提供设计支持，按项目收费

典型案例：市级逆变器代理
- 年销售：5MW（约100万营业额）
- 毛利率：8-12%
- 净利润：5-8万元
- 附加值：积累客户资源，可拓展EPC业务
```

**准入门槛**
- 初始投资：50-200万元（首批进货款）
- 仓储要求：100-300㎡仓库
- 流动资金：能支撑30-60天账期
- 区域要求：厂家设定的任务量

**适合人群**
- 有渠道资源（认识大量安装商）
- 资金周转能力强
- 熟悉供应链管理

#### 赛道四：光伏电站运维服务商

**业务定位**
- 为已建成电站提供专业运维服务
- 保障发电量，延长设备寿命

**商业模式**
```
服务内容：
├─ 日常巡检：每月1-2次现场巡检
├─ 设备维护：组件清洗、电气检查
├─ 故障抢修：24小时响应，48小时修复
├─ 数据分析：发电量监测、异常预警
└─ 备件管理：常用备件储备与更换

收费模式：
模式1：按装机容量
- 收费标准：30-60元/kW/年
- 适用：小型分布式（<500kW）

模式2：按发电量
- 收费标准：0.02-0.03元/kWh
- 适用：大型地面电站

模式3：效果对赌
- 保证发电量：设计值的95%
- 超额部分：业主与运维商五五分成
- 适用：高价值电站

项目案例：运维100个户用电站
- 平均每户：5kW
- 总装机：500kW
- 年服务费：500kW × 50元/kW = 2.5万元
- 成本：人工1.5万元/年，车辆及备件0.5万元
- 净利润：0.5万元（利润率20%）
- 规模化效应：管理500kW与5MW成本差不大
```

**准入门槛**
- 初始投资：10-30万元（设备、车辆）
- 技术要求：1-2名电气工程师
- 服务半径：100公里内（控制交通成本）
- 监控系统：自建或采购第三方平台

**适合人群**
- 有光伏技术背景
- 本地化服务（运维重在快速响应）
- 可与EPC业务协同（老客户转化）

#### 赛道五：光伏创新应用开发商

**新兴方向**

**5.1 光储充一体化**
```
业务场景：
- 停车场 + 光伏车棚 + 储能 + 充电桩
- 解决：用电高峰、电网容量不足问题

投资模型（典型项目）：
- 光伏：500kW，150万元
- 储能：500kWh，100万元
- 充电桩：20个，60万元
- 合计投资：310万元

收益来源：
├─ 光伏发电自用：节省电费 40万元/年
├─ 充电服务费：0.6元/kWh × 30万kWh = 18万元/年
├─ 峰谷套利：储能低价充、高价放 10万元/年
└─ 年总收益：68万元

静态回收期：4.6年
```

**5.2 农光互补/渔光互补**
```
模式：
- 下层：种植（如中药材、菌菇）或养殖（水产）
- 上层：光伏发电

优势：
├─ 土地双重利用，提高单位面积收益
├─ 光伏遮阴，降低夏季作物/水产温度
├─ 符合乡村振兴政策，易获批文
└─ 农业+电力双重收入

挑战：
- 需要农业专业知识
- 组件安装高度要求（农机作业空间）
- 部分遮挡影响发电量（需优化设计）
```

**5.3 BIPV（光伏建筑一体化）**
```
产品形态：
- 光伏瓦、光伏幕墙、光伏遮阳板
- 替代传统建材，同时发电

市场机会：
- 新建建筑强制绿色建筑标准
- 光伏成为建筑标配（类似外墙瓷砖）

商业模式：
├─ 与建筑设计院合作：嵌入设计方案
├─ 与建材商合作：渠道共享
└─ 高端项目定制：如地标建筑、展馆

附加值：
- 单价高于常规光伏（4-8元/瓦）
- 利润率：15-25%
```

**5.4 移动能源（便携式光伏）**
```
应用场景：
- 户外露营电源
- 应急救灾供电
- 偏远地区供电（牧民、哨所）

产品案例：
- 500W便携式光伏发电箱
  ├─ 折叠太阳能板：500W
  ├─ 储能电池：1kWh
  ├─ 逆变器：纯正弦波输出
  └─ 售价：3000-5000元

销售渠道：
- 电商平台（京东、淘宝）
- 户外用品店
- 政府应急采购
```

### 9.1.3 差异化竞争策略

光伏行业竞争激烈，必须找到差异化定位：

#### 策略1：垂直行业专家

**案例：专注冷库光伏系统**
```
差异化点：
├─ 深度了解冷库能耗特征（24小时高负荷）
├─ 优化方案：高自用比例（>90%）
├─ 行业人脉：冷链物流协会、大型超市采购
└─ 专业形象：冷库节能白皮书、行业峰会演讲

竞争优势：
- 客户获取成本低（口碑传播）
- 溢价能力强（专业方案值得更高价格）
- 复购率高（客户有多个冷库）
```

**其他可选垂直行业：**
- 医院光伏系统（医疗建筑绿色认证需求）
- 学校光伏系统（教育局集中采购）
- 加油站光伏系统（中石化等连锁系统）
- 酒店光伏系统（连锁品牌标准化复制）

---

#### 策略2：极致服务体验

**服务承诺示例：**
```
"7天快装承诺"
├─ 签约后7天内完成安装并网
├─ 超期每天赔偿500元
└─ 适用：100kW以下项目

"发电量保险"
├─ 承诺年发电量不低于设计值95%
├─ 不足部分按0.5元/kWh补偿
└─ 由第三方保险公司承保

"24小时故障响应"
├─ 报修后4小时内到场
├─ 24小时内恢复发电
└─ 配备应急替换设备（逆变器）
```

---

#### 策略3：技术创新引领

**创新方向：**
```
智能运维系统：
├─ 无人机自动巡检（热成像找隐裂）
├─ AI故障诊断（提前预警设备问题）
├─ 自动清洗机器人（降低人工成本）
└─ 客户手机App（随时查看收益）

高效组件+优化设计：
├─ 采用最新N型组件（效率23%+）
├─ 优化器防遮挡（组件级MPPT）
├─ 柔性支架（适应复杂屋顶）
└─ 发电量提升5-10%（强卖点）
```

---

#### 策略4：金融产品创新

**创新融资方案：**
```
"零首付"方案：
- 客户不出钱，投资方垫资
- 用节省的电费分期付款
- 6-8年付清，设备归客户

"收益权质押"贷款：
- 帮客户对接银行
- 用电站未来收益做质押
- 客户只需付30%，银行贷70%
- 还款来源就是发电收益

"电费托管"模式：
- 客户把电费委托给我们管理
- 我们保证总费用降低25%
- 多余的收益我们拿一部分
```

### 9.1.4 商业计划书要点

创业前必须写清楚的关键问题：

#### 1. 市场机会（Opportunity）
```
用数据说话：
- 目标市场规模：XX省工商业屋顶资源XX万㎡
- 市场增长率：年复合增长率30%
- 渗透率低：当前安装率<5%，空间巨大
- 政策支持：整县推进、碳中和目标

痛点与解决方案：
痛点1：企业电费高，侵蚀利润
→ 解决：光伏降低电费20-30%

痛点2：企业缺资金投资光伏
→ 解决：EMC模式，零投资

痛点3：安装后缺乏专业运维
→ 解决：25年全生命周期服务
```

#### 2. 竞争分析（Competition）
```
竞争格局：
├─ 全国性大型企业：天合、正泰等（占20%）
│   优势：品牌强、供应链优
│   劣势：本地化服务不足、响应慢
│
├─ 本地中小型企业：10-20家（占60%）
│   优势：灵活、关系好
│   劣势：资质弱、技术落后、服务参差
│
└─ 我们的定位：（目标占5-10%）
    差异化：垂直行业专家 + 极致服务 + 金融创新

竞争壁垒：
- 客户资源（签约客户案例库）
- 供应商关系（账期、价格优势）
- 团队能力（经验丰富的技术+销售团队）
- 品牌口碑（行业内推荐）
```

#### 3. 财务预测（Financials）

**三年财务预测（工商业EPC示例）**
```
第一年（启动期）：
├─ 目标：完成10个项目，共2MW
├─ 营收：700万元
├─ 成本：630万元
│   ├─ 设备及施工：560万元
│   ├─ 人员工资：40万元
│   └─ 营销及管理：30万元
├─ 净利润：70万元（利润率10%）
└─ 现金流：微亏20万元（账期影响）

第二年（成长期）：
├─ 目标：完成25个项目，共5MW
├─ 营收：1750万元
├─ 净利润：210万元（利润率12%，规模效应）
└─ 现金流：正100万元（上年项目回款）

第三年（稳定期）：
├─ 目标：完成40个项目，共8MW
├─ 营收：2800万元
├─ 净利润：420万元（利润率15%）
├─ 附加：运维收入50万元（前两年项目）
└─ 现金流：正300万元
```

#### 4. 团队介绍（Team）

```
创始人1：张三（CEO）

- 主导过50MW光伏电站建设
- 本地人脉资源丰富，认识100+企业主

创始人2：李四（CTO）
- 注册电气工程师，985硕士

- 对新技术敏感（N型组件、储能）

核心团队：
- 销售总监：5年工业品销售经验
- 施工总监：国家电网退休工程师
- 财务总监：注册会计师
```

#### 股权分配原则

**案例：三人合伙**
```
方案1：贡献法
├─ 技术合伙人：40%（核心竞争力）
├─ 销售合伙人：35%（带来业务）
└─ 运营合伙人：25%（后台支持）

方案2：资源+资金法
├─ 出资50万：30%
├─ 提供客户资源（价值估30万）：20%
├─ 全职投入（年薪20万，3年=60万）：50%

股权设计要点：
1. 创始人股权≥51%（控制权）
2. 预留期权池10-15%（激励员工）
3. 设置成熟期（4年，每年25%）
4. 设离职回购条款（避免僵局）
```

#### 合伙人协议核心条款

```
必备条款：
1. 股权比例及出资方式
   - 现金出资
   - 资源折价
   - 技术入股

2. 分工与职责
   - 谁负责什么（明确边界）
   - 重大决策机制（如超10万元支出需全体同意）

3. 利润分配
   - 年度分红比例
   - 再投资比例

4. 退出机制
   - 主动退出：按净资产回购
   - 被动退出（如严重失职）：按折扣回购
   - 竞业禁止：离职后2年内不得从事同业

5. 争议解决
   - 先协商
   - 再仲裁（避免诉讼伤害公司）
```

## 9.4 融资策略与公司运营管理

### 9.4.1 创业融资全周期规划

#### 阶段1：种子期融资（0-100万）

**资金来源：**
```
来源1：创始人自有资金（最优）
- 金额：30-50万
- 优势：无股权稀释，决策快
- 用途：公司注册、首批设备采购、3个月运营费用

来源2：亲友借款
- 金额：20-50万
- 方式：写借条，约定利息（如年化8%）
- 避免：股权换钱（亲友后期容易产生纠纷）

来源3：天使投资人
- 金额：50-200万
- 股权稀释：10-20%
- 寻找途径：
  ├─ 创业大赛（如"创青春"）
  ├─ 天使投资平台（如天使汇）
  └─ 本地企业家俱乐部

投资人看什么：
- 团队：创始人背景、互补性
- 市场：赛道够不够大、增速如何
- 模式：能否规模化复制
- 牵引力：是否已有客户或项目

如何估值：
- 种子期公司：200-500万估值
- 投资100万，给20%股权
- 投后估值500万
```

---

#### 阶段2：天使轮融资（100-500万）

**时机：**
```
达到以下里程碑时：
├─ 完成10个以上付费项目
├─ 年营收突破500万
├─ 商业模式跑通
└─ 需要资金快速扩张

资金用途：
├─ 团队扩充：销售团队从2人→8人
├─ 市场推广：品牌建设、线上获客
├─ 流动资金：缓解垫资压力
└─ 系统建设：CRM、项目管理系统
```

**融资渠道：**
```
1. 专业天使投资机构
   - 如：真格基金、险峰长青、梅花创投
   - 关注：新能源、节能环保赛道

2. 产业投资人
   - 如：光伏上市公司（隆基、天合等）
   - 战略价值：除了钱，还带来资源

3. 政府引导基金
   - 各地创业投资引导基金
   - 优势：估值合理、不抢控制权
   - 劣势：决策慢、流程复杂

估值方法：
- 收入倍数法：年营收500万 × 2-3倍 = 1000-1500万估值
- 融资300万，稀释20-25%股权
```

**时机：**
```
达到以下条件：
├─ 年营收：3000万以上
├─ 净利润：300万以上
├─ 增长率：年增长100%以上
├─ 商业模式：已在多个区域验证
└─ 用途：跨区域扩张

资金用途：
├─ 区域扩张：开设5-10个城市分公司
├─ 品牌建设：全国性推广
├─ 技术研发：智能运维系统、数字化工具
└─ 团队建设：引入高管（副总裁级）
```

**融资渠道：**
```
1. VC机构（风险投资）
   - 红杉、经纬、IDG等
   - 看重：团队、增长、市场空间

2. 产业基金
   - 国家电投、三峡资本等
   - 战略协同价值

3. 政府产业引导基金
   - 各地新能源产业基金

估值：
- 利润倍数法：净利润300万 × 15-25倍 = 4500万-7500万
- 融资2000万，稀释20-30%
```

**供应链金融**

```
产品：应收账款融资

产品：订单融资
- 拿到大额订单但缺资金采购

适用场景：
```

```
适用条件：

- 发电收益稳定可预测
- 规模：至少1亿元

案例：

→ 未来20年预计发电收益10亿元
→ 发行ABS融资5亿元
```

### 9.4.2 财务管理与成本控制

#### 财务核心指标监控

**仪表盘（每月必看）：**
```
收入指标：
├─ 当月签约额：目标vs实际
├─ 当月确认收入：完工项目收入
└─ 回款额：现金入账

成本指标：
├─ 设备采购成本占比：控制在60-65%
├─ 人工成本占比：控制在15-20%
└─ 管理费用率：控制在8%以下

利润指标：
├─ 毛利率：目标30-35%
├─ 净利率：目标10-15%
└─ 单项目利润：每个项目独立核算

现金流指标（最重要！）：
├─ 账上现金余额：≥3个月运营费用
├─ 应收账款账龄：90%在6个月内
└─ 应付账款账龄：不超过合同约定

预警线：
- 现金流断裂：账上余额<1个月费用
- 应收恶化：6个月以上应收占比>30%
- 利润率低：净利率<5%
```

---

#### 成本控制策略

**策略1：采购成本优化**
```
招数1：集中采购
- 联合其他公司一起采购
- 提高议价能力

招数2：季度锁价
- 年初与供应商谈好全年价格
- 避免价格波动风险

招数3：多家比价
- 至少3家供应商报价
- 但不能只看价格，也看质量和服务

招数4：缩短账期
- 现款采购争取3-5%折扣
- 计算：如果年化资金成本<20%，值得

招数5：库存最小化
- 项目制采购，不囤货
- 只备少量快消品（线缆、接插件）
```

**策略2：人力成本优化**
```
招数1：多劳多得
- 销售、施工等岗位高提成低底薪
- 避免养闲人

招数2：外包非核心
- 施工班组：项目制外包
- 财务：代账公司
- 法务：年度顾问而非全职

招数3：期权替代高薪
- 核心骨干给期权
- 降低当期现金支出

招数4：灵活用工
- 旺季临时工
- 淡季人员精简
```

**策略3：管理费用控制**
```
办公成本：
- 初期无需高档写字楼
- 共享办公或工业园区办公
- 控制在3000-5000元/月

车辆成本：
- 初期租车或员工用车补贴
- 规模化后再购置

营销费用：
- 初期以低成本方式（微信、抖音）
- 避免大规模广告投放（ROI低）
- 重点：做好口碑，靠转介绍

差旅费：
- 原则：就近开发客户
- 跨地区项目要算清楚成本
```

### 9.4.3 风险管理与合规运营

#### 法律风险防范

**风险点1：合同纠纷**
```
常见纠纷：
- 客户拖欠款项
- 发电量未达预期客户要求赔偿
- 屋顶漏水责任归属

防范措施：
1. 规范合同文本
   - 使用律师审核过的标准合同
   - 明确双方权利义务
   - 约定违约责任

2. 关键条款设计
   - 付款节点与比例（30-30-30-10）
   - 发电量保证（设计值的90%，低于补偿）
   - 不可抗力免责（台风、冰雹等）
   - 争议解决（仲裁优于诉讼）

3. 过程留痕
   - 施工前：屋顶现状拍照
   - 施工中：每日拍照记录
   - 验收时：客户签字确认
   - 运营中：发电数据月报
```

**风险点2：资质合规**
```
必须持有的证照：
├─ 营业执照
├─ 电力工程施工资质（或挂靠）
├─ 安全生产许可证（施工类）
└─ ISO体系认证（大项目需要）

风险：
- 无证施工：罚款、停业整顿
- 挂靠风险：挂靠公司卷款跑路

合规建议：
- 规模小时先挂靠
- 年营收超2000万后自己办资质
- 专人负责资质年检、续期
```

**风险点3：安全生产**
```
重点风险：
- 高空坠落（屋顶施工）
- 触电事故（电气作业）
- 火灾（直流拉弧）

防范措施：
1. 制度建设
   - 安全操作规程
   - 三级安全教育
   - 安全检查制度

2. 安全投入
   - 安全带、安全帽等防护用品
   - 高空作业保险
   - 灭火器等消防器材

3. 应急预案
   - 触电急救预案
   - 火灾应急预案
   - 定期演练

4. 保险配置
   - 雇主责任险：保员工工伤
   - 公众责任险：保第三方损失
   - 工程一切险：保施工期风险
```

---

#### 财务合规

**税务合规：**
```
税种及税率：
├─ 增值税：6%（建筑服务）或13%（设备销售）
│   - 注意：一般纳税人可以抵扣进项
│   - 小规模纳税人：1%（疫情优惠，后续可能调整）
│
├─ 企业所得税：25%（一般企业）
│   - 高新技术企业：15%
│   - 小微企业：实际5-10%
│
├─ 附加税：增值税的12%（城建税、教育费附加）
│
└─ 个人所得税：员工工资代扣代缴

税务风险：
- 虚开发票：严禁！
- 隐瞒收入：容易被查
- 成本无发票：无法税前扣除

合规建议：
1. 账务规范
   - 聘请专业会计或代账公司
   - 收入、成本及时入账
   - 发票管理严格

2. 税务筹划
   - 设备采购从一般纳税人买（可抵扣）
   - 申请高新企业（15%税率）
   - 研发费用归集（加计扣除75%）

3. 按时申报
   - 每月增值税申报
   - 每季度所得税预缴
   - 每年汇算清缴
```

## 📊 创业实战模拟演练

### 模拟场景：三个创始人准备创业

**背景：**
```
创始人A（老王）：
- 40岁，原电力公司工程师
- 有技术，有项目经验
- 出资30万

创始人B（小李）：
- 35岁,原工业设备销售
- 有客户资源，能谈客户
- 出资20万

创始人C（老张）：
- 38岁，原企业财务总监
- 懂财务，擅长管理
- 出资20万
```

**第一步：选方向**
```
讨论结果：
- 方向：工商业分布式光伏EPC
- 理由：
  ✓ 小李有10多个工厂客户资源
  ✓ 老王技术能力强，能设计能施工
  ✓ 投资不大，70万启动资金够
  ✓ 政策支持，市场空间大
```

**第二步：做计划**
```
第一年目标：
- 签约：10个项目，共1.5MW
- 营收：500万
- 净利润：50万（10%净利率）

资源配置：
- 办公室：租园区办公室，3000元/月
- 人员：3个创始人+2个销售+1个施工队长+外包施工队
- 供应商：谈下来隆基组件和锦浪逆变器代理

启动资金分配：
├─ 注册及办公：5万
```

## 🎯 创业成功的10大关键要素

### 1. 选对赛道
- 市场够大、增速够快
- 政策支持、符合趋势
- 自己有资源或优势

### 2. 组对团队
- 创始团队互补（技术+销售+运营）
- 找到靠谱的人
- 股权激励绑定

### 3. 找准定位
- 差异化（不跟大公司正面竞争）
- 垂直行业专家或区域深耕
- 用服务和效率赢得客户

### 4. 控制节奏
- 不盲目扩张
- 现金流为王
- 一步一个脚印

### 5. 重视客户
- 客户满意度决定口碑
- 口碑决定获客成本
- 老客户转介绍是最好的渠道

### 6. 修炼内功
- 标准化流程
- 数字化工具
- 持续提升效率

### 7. 整合资源
- 供应链资源
- 客户资源
- 跨界资源
- 政府资源

### 8. 财务健康
- 现金流>利润
- 应收账款管理
- 成本严格控制

### 9. 合规经营
- 资质齐全
- 税务规范
- 安全第一

### 10. 不断学习
- 技术迭代快
- 政策变化快
- 保持敏锐度
- 终身学习

## 💡 写给创业者的话

光伏创业，是一场马拉松，不是百米冲刺。

**会遇到的困难：**
- 客户拖款，现金流紧张
- 价格战，利润微薄
- 招人难，留人难
- 政策变化，项目受影响
- 竞争对手挖墙脚

**但也会有的收获：**
- 第一个项目并网的成就感
- 客户感谢信
- 团队从3人到30人的成长
- 从0到1000万营收的突破
- 为社会节能减排做出贡献

**记住：**
1. 保持初心：为什么创业？是为了财富自由，还是为了证明自己，还是为了改变世界？初心会支撑你度过最艰难的时刻。

2. 学会放弃：不是所有客户都要接，不是所有机会都要抓。学会说"不"，聚焦核心业务。

3. 拥抱变化：光伏行业变化太快了。组件从P型到N型,逆变器从集中式到组串式，储能正在成为标配...保持开放心态，拥抱变化。

4. 善待团队：员工是最宝贵的资产。多分享,多培养，让大家跟着你有成长、有收获。

5. 敬畏市场：永远保持对市场的敬畏，对客户的敬畏，对专业的敬畏。骄兵必败。

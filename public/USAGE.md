# 快速使用指南

## 🚀 5分钟快速上手

### 第一步：打开应用
- **本地使用**：直接用浏览器打开 `public/index.html` 文件
- **在线访问**：部署后通过URL访问

### 第二步：选择场景
应用提供三种预设场景：
- 🟠 **保守场景**：低估收入，高估成本
- 🔵 **标准场景**：行业平均水平（推荐）
- 🟢 **乐观场景**：高订单量，低成本

### 第三步：调整参数
根据实际情况调整以下关键参数：

**必填参数**
1. 投放站点数量（如：100个）
2. 每站点日均订单（如：2.2单）
3. 单次租赁价格（如：15元）
4. 单站设备成本（如：2500元）
5. 场地分成比例（如：20%）

**选填参数**（点击"高级参数"展开）
- 物流、软件、营销等成本
- 日常运营费用明细

### 第四步：查看结果
系统会实时计算并展示：

**关键指标卡片**
- ⏱️ 投资回收期（月）+ 评级
- 📈 年化ROI百分比
- 💰 月净利润
- ⚖️ 盈亏平衡点

**详细分析标签**
1. **概览** - 所有数据一览
2. **收入分析** - 营业额和分成
3. **成本分析** - CAPEX和OPEX
4. **利润分析** - 利润率和现金流
5. **可视化** - 4个交互图表

### 第五步：导出报告
点击底部导出按钮：
- 📊 **Excel表格** - 详细数据（已实现）
- 📄 **PDF报告** - 专业分析（开发中）
- 🖼️ **图片分享** - 关键指标（开发中）
- 🔗 **链接分享** - 带参数URL

## 📱 移动端使用
- 自适应布局，手机完美展示
- 关键指标优先显示
- 标签页切换查看详细数据

## 💡 使用技巧

### 1. 对比不同方案
1. 设置第一套参数并导出Excel
2. 切换场景或修改参数
3. 再次导出，对比两份数据

### 2. 敏感性分析
尝试调整单一参数，观察ROI变化：
- 订单量 ±20%
- 租赁价格 ±10%
- 场地分成 ±5%

### 3. 找到最优策略
- **订单量**：最关键因素，优先选址
- **租赁价格**：需平衡用户接受度
- **场地分成**：影响净利率

### 4. 风险评估
- 保守场景回收期 < 24个月 → 可行
- 标准场景年化ROI > 50% → 优秀
- 乐观场景仅作参考

## ⚠️ 注意事项

### 参数设置建议
1. **订单量**：参考同类场地数据，保守预估
2. **价格**：调研市场行情，不宜过高
3. **分成比例**：与场地方充分协商
4. **运营成本**：预留10-20%缓冲

### 常见错误
❌ 过度乐观估计订单量
❌ 忽略隐性成本（人力、维护）
❌ 未考虑季节性波动
❌ 未预留营销推广预算

✅ 多场景对比分析
✅ 保守预估财务指标
✅ 预留应急资金
✅ 定期复盘调整

## 🎓 计算公式说明

### 投资回收期
```
回收期(月) = 初始投资 ÷ 月净利润
```

### 年化ROI
```
年化ROI = (年净利润 ÷ 初始投资) × 100%
```

### 净利率
```
净利率 = (净利润 ÷ 营业额) × 100%
```

### 盈亏平衡点
```
平衡点(单/天) = (日运营成本 + 日分成金额) ÷ 单次价格
```

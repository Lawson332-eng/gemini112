# 共享充电宝ROI计算器

一款专业的共享充电宝投资回报率(ROI)在线计算工具，帮助投资者和运营商快速评估项目可行性。

## ✨ 功能特性

### 🎯 核心功能
- **精准计算引擎** - 基于行业标准的ROI计算模型
- **实时反馈** - 输入参数即时计算，300ms防抖优化
- **场景预设** - 保守/标准/乐观三种预设场景快速切换
- **多维度分析** - 收入、成本、利润全方位数据分析

- 投资回收期（天/月）
- 年化ROI百分比

- 盈亏平衡点

## 📖 使用说明

### 基础参数
1. **投放站点数量** - 计划投放的充电宝站点总数（1-10000）
2. **每站点日均订单** - 预估每个站点每天的租用次数（0.1-100）
3. **单次租赁价格** - 用户每次租用的费用（0.5-100元）
4. **单站设备成本** - 包括机柜和充电宝的硬件成本（100-50000元）
5. **场地分成比例** - 支付给场地方的收入分成（0-80%）

### 高级参数（点击展开）
- 单站物流成本
- 软件许可费
- 营销推广费
- 支付手续费率
- 供应商分成比例
- 日均人工成本
- 日均通信成本
- 日均系统维护费
- 充电宝损耗成本
- 预期运营天数

### 场景预设
- **保守场景** - 低订单量、高成本，适合风险厌恶型投资者
- **标准场景** - 行业平均水平，基于市场调研数据
- **乐观场景** - 高订单量、低成本，适合乐观预期

### 结果分析标签页
1. **概览** - 快速查看所有关键数据
2. **收入分析** - 营业额和分成明细
3. **成本分析** - CAPEX和OPEX详细构成
4. **利润分析** - 利润率和现金流分析
5. **可视化** - 4个交互式图表

## 🎯 计算公式

### 收入计算
```
日总订单数 = 站点数量 × 每站点日均订单数
日营业额 = 日总订单数 × 单次租赁价格
```

### 成本计算
```
初始投资(CAPEX) = (单站设备成本 + 单站物流成本) × 站点数量
                   + 软件许可费 + 营销推广费

日运营成本(OPEX) = 日均人工成本 + 日均通信成本
                   + 日均系统维护费 + (充电宝损耗成本 × 日总订单数)
```

### 利润计算
```
日毛收入 = 日营业额 × (1 - 场地分成比例 - 供应商分成比例 - 支付手续费率)
日净利润 = 日毛收入 - 日运营成本
```

### 关键指标
```
投资回收期(月) = 初始投资 ÷ 日净利润 ÷ 30
年化ROI(%) = (年净利润 ÷ 初始投资) × 100%
毛利率(%) = (日毛收入 ÷ 日营业额) × 100%
净利率(%) = (日净利润 ÷ 日营业额) × 100%
```
